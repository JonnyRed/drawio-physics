# Installation Guide - Classical Mechanics Draw.io Library

## Quick Start (3 Steps)

### Step 1: Download the Library
- Clone this repository or download `classical-mechanics.xml`

### Step 2: Open Draw.io
- Go to [draw.io](https://www.draw.io) or open the desktop app
- Create a new diagram or open an existing one

### Step 3: Import the Library
- Click **More Shapes** (bottom left panel)
- Click **Edit Shapes**
- Click **Import**
- Select `classical-mechanics.xml`
- Click **Import**

The library is now available in your shape panel!

---

## Detailed Installation Methods

### Method 1: Online draw.io (Recommended for Beginners)

1. **Open draw.io**
   - Navigate to https://www.draw.io
   - Click **Create New Diagram** or open an existing one

2. **Access Shape Library Manager**
   - In the left panel, look for **More Shapes** button (or scroll down in shapes panel)
   - Click the gear icon ⚙️ or **Edit Shapes**

3. **Import Custom Shapes**
   - Click **Import** or the **+** button
   - A file browser will open
   - Navigate to and select `classical-mechanics.xml`
   - Click **Open**

4. **Verify Import**
   - The "Classical Mechanics" library will appear in your shapes panel
   - You'll see categories for:
     - Force Representations
     - Mechanical Components
     - Free Body Diagrams
     - Kinematic Indicators
     - Constraints & Supports
     - Common Systems
     - Physics Notation

5. **Start Using**
   - Expand any category by clicking it
   - Drag shapes onto your canvas
   - Double-click to edit text
   - Right-click to access formatting options

---

### Method 2: Draw.io Desktop App

1. **Install Draw.io Desktop**
   - Download from [draw.io releases](https://github.com/jgraph/drawio-desktop/releases)
   - Install for your OS (Windows, macOS, or Linux)

2. **Launch the Application**
   - Open Draw.io from your applications

3. **Import Library**
   - File → New → Create new diagram
   - Left panel: Click **More Shapes**
   - Click **Edit Shapes**
   - Click **Import** button
   - Browse to `classical-mechanics.xml`
   - Click **Open**

4. **Persistent Library**
   - Once imported, the library is saved locally
   - Available for all future diagrams
   - No need to re-import

---

### Method 3: GitHub-Hosted Library (Advanced)

For organizations wanting centralized library management:

1. **Host on GitHub**
   - Fork this repository
   - Keep `classical-mechanics.xml` in main directory

2. **Import in Draw.io**
   - More Shapes → Edit Shapes → Import
   - Use the raw GitHub URL:
     ```
     https://raw.githubusercontent.com/YOUR-USERNAME/drawio-physics/main/classical-mechanics.xml
     ```

3. **Share with Team**
   - Team members can import using the same URL
   - Updates automatically when you push changes

---

### Method 4: Standalone XML Import

1. **Copy XML Content**
   - Open `classical-mechanics.xml` in a text editor
   - Copy entire contents

2. **In Draw.io**
   - More Shapes → Edit Shapes → Create
   - Paste XML content directly
   - Click **Create**

## Troubleshooting

### "Import Failed" Error
- **Issue**: File format not recognized
- **Solution**: 
  - Ensure file is named `classical-mechanics.xml` (not `.txt`)
  - Check file is valid XML (open in browser to verify)
  - Try dragging file directly onto canvas instead

### Shapes Don't Appear
- **Issue**: Library imported but no shapes visible
- **Solution**:
  - Scroll down in "More Shapes" panel
  - Look for "Classical Mechanics" category
  - Click to expand if collapsed
  - Try File → Recent → Reload

### Color/Style Issues
- **Issue**: Shapes appear different than expected
- **Solution**:
  - Click shape → Format panel (right side)
  - Adjust Fill, Stroke, Line Width as needed
  - Shapes are fully customizable

### Import on Mobile
- **Issue**: Can't import on iPad/mobile
- **Solution**:
  - Use desktop browser version (more.draw.io works better)
  - Or use Draw.io apps on mobile (limited functionality)
  - Alternative: Use online version on tablet with keyboard

## File Locations

After import, shape library files are stored at:

**Windows:**
```
C:\Users\[YourUsername]\AppData\Roaming\draw.io\custom-shapes.xml
```

**macOS:**
```
~/Library/Application Support/draw.io/custom-shapes.xml
```

**Linux:**
```
~/.config/draw.io/custom-shapes.xml
```

**Online draw.io:**
- Stored in browser localStorage (specific to each device)
